Open Source Project

Cancer Genomics

I had the opportunity to participate in the analysis of ESCC samples using three transcriptomic platforms. Some of the analyses I've been involved in include:

Detecting doublets and multiplets
Correcting batch effects between scRNA-seq and snRNA-seq

Python

Bash

I am capable of carrying out basic tasks using shell scripting and utilizing Git.

Java

VcfFormatParsedTxt.jar: I have developed a straightforward Java program that parses the FORMAT column of a VCF file and converts it into a TXT file.

Kotlin

PortfolioWebsite: I utilized Kotlin to build this website.

bamscope: I have developed software using Kotlin that employs samtools to extract reads from a BAM file. It then generates plots displaying stacked bases at a designated position on a chromosome, with each base represented in a distinct color. Currently, I am in the process of addressing and fixing bugs in the software.
